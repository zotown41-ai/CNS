手机端查看建议

  1. 开启 GitHub Pages 后，直接打开仓库对应页面网址。
  2. 手机浏览器首次打开本页后，可使用“添加到主屏幕”，以后像 App 一样直接点开。
  3. 日常只看最新内容时，优先点击“查看日报页面”或“查看周报页面”。
  4. 如果需要复制、检索或保存原始文本，再打开对应的 Markdown 文件。
