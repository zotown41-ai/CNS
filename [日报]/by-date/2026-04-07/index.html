Sharkmer：利用计算机模拟PCR重用PCR引物进行靶向基因组组装 / Sharkmer: repurposing PCR primers for targeted genome assembly using in silico PCR

  • 日期：2026-04-07
  • 作者：未提供
  • 文章类型：未提供
  • 研究问题：如何利用既有PCR引物序列与原始测序读段，通过计算机模拟PCR实现特定基因组区域的快速、定向组装，以支持常规的物种鉴定、污染识别和系统发育分析？

发现/highlights：

  1. 提出基于体外PCR引物的计算机模拟PCR方法，用于从原始测序读段中定向组装特定基因组区域。
  2. 在 sharkmer 中实现了以正向引物为种子、以反向引物为终止条件的靶向 de Bruijn 图组装流程。
  3. 该方法可用于数据集物种身份验证、污染检测以及基于原始序列数据的快速系统发育分析。

摘要：本文介绍了一种计算机模拟PCR（in silico PCR，sPCR）方法，可利用原始测序读段组装由PCR引物跨越的特定基因组区域。该方法使用户能够快速分离公共基因序列档案中较为丰富的目标区域，并利用数十年来实验室PCR引物优化工作的积累。作者在 sharkmer 中实现了该方法，将其设计为一种靶向 de Bruijn 图组装器：以前向引物序列作为种子，以反向引物序列作为终止条件。该方法适用于多种常规任务，包括验证数据集的物种身份、识别污染物，以及基于原始序列数据快速构建系统发育关系。sharkmer 采用 Rust 编写，代码、安装与使用说明、测试及其他资源可在 GitHub 仓库和 Zenodo 获取，也可通过 bioconda 安装。

链接：https://academic.oup.com/bioinformatics/advance-article/doi/10.1093/bioinformatics/btag163/8626953?rss=1
